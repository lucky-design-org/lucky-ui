- ✂️ 完善的代码提示

## 使用

安装lucky-ui

```cmd
# 选择一个你喜欢的包管理器

# NPM
$ npm install lucky-ui --save

# Yarn
$ yarn add lucky-ui --save

# pnpm
$ pnpm add lucky-ui --save
```
